# VPN

## 오픈소스 설치형 VPN 솔루션

| 솔루션       | 특징                                     |
|--------------|------------------------------------------|
| [WireGuard]  | 커널 내장, 최고 성능, ~4,000줄 코드      |
| [WG-Easy]    | WireGuard + 웹 UI, Docker 한 줄로 시작   |
| [Headscale]  | Tailscale 오픈소스 대체, 같은 클라이언트  |
| [NetBird]    | 제로 트러스트, P2P, SSO, 웹 관리 UI      |
| [OpenVPN]    | 가장 오래되고 검증됨, 커스터마이징 최고   |
| [AmneziaVPN] | 검열/DPI 우회 특화, 서버 자동 설치       |
| [Algo VPN]   | 클라우드 자동 배포, 관리 부담 최소       |
| [Tailscale]  | WireGuard 기반 SaaS, 설정 거의 불필요   |
| [Pritunl]    | OpenVPN/WireGuard 통합 서버              |

[WireGuard]: https://www.wireguard.com/
[WG-Easy]: https://github.com/wg-easy/wg-easy
[Headscale]: https://github.com/juanfont/headscale
[NetBird]: https://github.com/netbirdio/netbird
[OpenVPN]: https://openvpn.net/
[AmneziaVPN]: https://github.com/amnezia-vpn/amnezia-client
[Algo VPN]: https://github.com/trailofbits/algo
[Tailscale]: ./tailscale.md
[Pritunl]: ./pritunl.md
