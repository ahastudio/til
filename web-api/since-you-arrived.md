# Since You Arrived — 브라우저가 드러내는 것들

<https://sinceyouarrived.world/taken>

## 요약

Matt가 만든 인터랙티브 웹페이지로, 방문자의 브라우저가 JavaScript API만으로 얼마나 많은 정보를 자동으로 노출하는지 시각적으로 보여준다.
익스플로잇이나 악성코드 없이 IP 기반 위치, 기기 사양, 폰트 핑거프린트, 캔버스 핑거프린트를 수집한다.
페이지는 두 개의 익명 이벤트만 전송하고 쿠키나 localStorage를 일절 사용하지 않는다.
탭을 닫으면 모든 정보를 "잊는다"고 명시한다.

## 분석

### 수집 정보 목록

- **위치**: IP 주소 → ip-api.com으로 도시·ISP 파악
- **기기**: 화면 크기, GPU, CPU 코어 수, 배터리 잔량, 언어, 시간대
- **브라우저 고유값**: 설치된 폰트 조합, 캔버스 렌더링 픽셀값

### 핑거프린팅 메커니즘

**글꼴 핑거프린팅**은 숨겨진 텍스트를 렌더링한 뒤 너비를 측정해 어떤 폰트가 설치됐는지 추론한다.
**캔버스 핑거프린팅**은 보이지 않는 캔버스에 이미지를 그리고 픽셀값을 해시해 브라우저·GPU 조합을 식별한다.
배터리 API는 잔량과 방전 속도 조합만으로 30분간 추적에 충분한 고유성을 제공한다고 페이지는 설명한다.
이 모든 기법은 EFF의 "Cover Your Tracks"를 지적 토대로 삼는다.

### 투명성 설계

클립보드 접근, favicon 기반 로그인 감지 같은 "더 침습적인 기법"은 의도적으로 제외했다고 밝힌다.
코드 전체가 공개되어 있으며 생성형 AI는 사용하지 않았다.

## 비평

[GeekNews 댓글](https://news.hada.io/topic?id=29313)에서 한 사용자가 Apple Private Relay를 사용하자 위치가 수백 마일 벗어났다고 보고했다.
IP 기반 위치 정보의 한계를 실제로 드러낸 사례다.

시간대·언어 같은 "기본 기술 정보"가 실제로 문제가 되는 감시인지, 아니면 예상 범위의 기능인지에 대한 논쟁도 있었다.
페이지가 LLM 스타일의 극적인 문체로 상대적으로 평범한 추적 데이터를 과장한다는 비판도 나왔다.
실제 광고 생태계는 이보다 훨씬 정교한 추적을 이미 수행하고 있기 때문이다.

## 인사이트

브라우저는 별도 허가 없이 방대한 정보를 노출한다.
"개인정보 보호"가 실제로 무엇을 막고 무엇을 막지 못하는지, 이 페이지는 기술적 사실만으로 설명한다.
VPN이나 Private Relay가 IP 위치를 가리더라도 캔버스·폰트 핑거프린트는 여전히 유효하다는 점이 핵심 교훈이다.
브라우저 핑거프린팅 저항이 IP 익명화보다 실질적 추적 방어에 더 중요할 수 있다.
