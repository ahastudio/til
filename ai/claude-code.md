# Claude Code

> **Claude Code**는 터미널에 상주하며 코드베이스를 이해하고,
> 일상적인 작업을 실행하고, 복잡한 코드를 설명하고, Git 워크플로를 처리하여
> 코딩 속도를 높여주는 에이전트형 코딩 도구입니다.
> 이 모든 작업은 자연어 명령어를 통해 수행됩니다.
