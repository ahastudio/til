# Korean Law MCP

- 원문: <https://github.com/chrisryugj/korean-law-mcp>

## 요약

대한민국 법령을 MCP(Model Context Protocol)로 검색·조회·분석할 수 있는
도구다. Claude 등 MCP를 지원하는 AI 에이전트에 연결하면 법령 데이터를
직접 질의할 수 있다.

국가법령정보센터 API를 기반으로 법령 본문, 조문 검색, 개정 이력 조회
기능을 제공한다.

## 핵심 기능

- 법령명·조문 키워드 검색
- 특정 조문 본문 조회
- 시행일·개정 이력 확인
- 관련 법령 연결 탐색

## 인사이트

### 1) 도메인 특화 MCP가 AI 실무 활용의 실질적 경로다

일반 LLM은 법령 최신성을 보장할 수 없다. 개정 시점 이후의 변경 사항은
학습 데이터에 없고, 조문 번호나 항·호 단위의 정확한 인용도 환각 위험이
크다.

Korean Law MCP는 공식 API를 직접 연결하므로 최신 법령을 정확한 출처와
함께 제공할 수 있다. "AI가 법을 안다"가 아니라 "AI가 법령 DB를 쓴다"는
구조적 전환이다.

### 2) 수직 특화 MCP는 전문직 도메인에서 더 빠르게 가치를 낸다

법률, 의료, 회계처럼 정확성이 곧 리스크인 분야는 범용 LLM만으로는 신뢰를
얻기 어렵다. 공식 데이터 소스와 연결된 MCP 레이어가 있으면 AI 보조
도구로서의 신뢰 기준을 충족할 수 있다.

변호사·법무팀이 직접 쓰는 도구가 되려면 법령 MCP 수준의 정확성이 최소
조건이다.

### 3) 한국어 법령 도메인에서 오픈소스 기여의 공백이 있다

영미권에는 법령 API 래퍼가 다수 존재하지만, 한국 법령을 AI 워크플로에
바로 연결하는 도구는 드물었다. 이 프로젝트는 그 공백을 채우는 동시에
MCP 생태계에서 한국어 도메인 데이터 연결의 선례를 만든다.

유사한 패턴으로 판례(대법원 종합법률정보), 규제(금융감독원 공시) 등
다른 공공 데이터 소스로 확장할 수 있다.
