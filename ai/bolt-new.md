# Bolt.new - AI Web App Builder

<https://bolt.new/>

StackBlitz가 만든 AI 기반 풀스택 웹 앱 빌더.
프롬프트를 입력하면 웹사이트, 앱, 프로토타입을 즉시
생성한다. 브라우저 안에서 개발, 미리보기, 배포까지
원스톱으로 처리할 수 있다.

## 핵심 기술

### WebContainers

StackBlitz가 7년간 개발한 브라우저 내 런타임 기술.
Node.js를 포함한 전체 개발 환경이 브라우저에서 실행된다.
클라우드 서버 없이 동작하므로 빠르고 비용이 낮다.

### AI 환경 제어

AI 모델이 파일 시스템, 노드 서버, 패키지 매니저,
터미널, 브라우저 콘솔 등 전체 환경을 제어한다.
앱 생성부터 배포까지 전체 라이프사이클을 처리한다.

### LLM 통합

Claude Sonnet, GPT 등 다양한 LLM을 선택할 수 있다.
2026년에는 Opus 4.6과 조절 가능한 추론 깊이(Reasoning
Depth)가 추가되었다.

## 주요 기능

- **프롬프트 기반 개발**: 자연어로 앱을 설명하면 동작하는
  프로젝트를 생성
- **풀스택 생성**: 프론트엔드, 백엔드, 데이터베이스, API
  엔드포인트를 한 번에 생성
- **브라우저 IDE**: 라이브 프리뷰, 파일 트리, 터미널을
  포함한 완전한 개발 환경
- **원클릭 배포**: Vercel, Netlify, Cloudflare 등으로
  즉시 배포
- **외부 연동**: Figma, GitHub, Supabase, Firebase,
  Stripe 등과 통합
- **오픈소스**: GitHub에 공개되어 있으며, bolt.diy로
  커스터마이징 가능

## 성장 스토리

StackBlitz는 2023년 말 연매출 $80,000에 불과했다.
성장이 정체되어 회사 폐쇄까지 논의되었다.

2024년 10월 4일 Bolt.new를 출시하면서 상황이 반전되었다.
6개월 만에 ARR $40M(약 560억 원)을 달성했다.
2025년 3월에는 100만 개 이상의 웹사이트가 Bolt로
배포되었다. 2025년 5월 기준 월 방문자 900만 명,
평균 체류 시간 22분을 기록했다.

## 요금제

토큰 기반 과금 체계를 사용한다.

| 플랜         | 가격     | 토큰           |
| ------------ | -------- | -------------- |
| Free         | 무료     | 제한적         |
| Pro          | $20/월   | 1,000만 토큰   |
| Pro+         | $50/월   | 3,200만 토큰   |
| Team         | $200/월  | 1억 2,000만    |
|              |          | 토큰           |

2025년 8월부터 호스팅, 도메인, 데이터베이스, 서버리스
함수, 인증, SEO, 결제, 분석 기능이 구독에 포함되었다.
코드 생성 도구에서 "빌드 + 운영 + 확장" 플랫폼으로
진화하고 있다.

## 강점

- **속도**: 아이디어에서 동작하는 앱까지 수 분이면 완성.
  해커톤, MVP, 클라이언트 데모에 적합
- **접근성**: 설치 없이 브라우저에서 바로 사용.
  비개발자도 활용 가능
- **코드 소유권**: 벤더 락인 없이 코드를 완전히 소유.
  언제든 외부로 내보낼 수 있다
- **UI 품질**: 깔끔하고 모던한 디자인이 기본 제공되어
  스타일링 시간을 절약

## 한계

- **복잡한 앱**: 대규모 프로젝트나 미션 크리티컬 앱에는
  안정성과 커스텀 코드가 부족
- **Fix-and-Break 사이클**: 한 문제를 고치면 다른 곳이
  깨지는 악순환이 발생할 수 있다
- **토큰 비용**: 디버깅 과정에서 토큰 소모가 누적되어
  비용이 급증할 수 있다
- **프롬프트 학습 곡선**: 모호한 프롬프트는 엉망인 결과를
  낳는다. 명확하고 단계적인 지시가 필요하다
- **협업 제한**: 기본적으로 1인 개발자 도구.
  팀 협업은 코드 내보내기 후 별도 Git 저장소를 사용해야
  한다

## 적합한 사용처

- 프로토타입과 MVP를 빠르게 만들고 싶은 솔로 개발자
- UI/UX 아이디어를 직접 검증하고 싶은 디자이너/PM
- 내부 도구, 대시보드를 신속히 구축해야 하는 스타트업
- 해커톤 참가자와 사이드 프로젝트
- 실제 앱을 만들며 배우고 싶은 입문자

## 부적합한 사용처

- 수백 명이 사용하는 대규모 엔터프라이즈 앱
- 복잡한 마이크로서비스 아키텍처
- 세밀한 아키텍처 설계와 커스텀 워크플로가 필요한 프로젝트
- 높은 성능과 신뢰성이 요구되는 시스템

## 인사이트

### 바이브 코딩 도구의 성숙

Bolt.new는 [바이브 코딩](./vibe-coding.md) 트렌드의
대표적인 구현체다. "코드를 잊고 결과에 집중"하는 접근이
실제 비즈니스에서 통한다는 것을 $40M ARR로 증명했다.

### 개발 도구의 플랫폼화

코드 생성에서 시작해 호스팅, 도메인, 인증, 결제까지
번들링하는 전략은 Vercel, Netlify의 경로를 따른다.
AI 코드 생성기가 결국 풀 플랫폼으로 수렴하는 패턴이
나타나고 있다.

### WebContainers의 가치

7년간의 기술 투자가 AI 시대에 빛을 발한 사례.
브라우저 내 런타임이라는 기반 기술이 없었다면
Bolt.new의 즉각적인 피드백 루프는 불가능했을 것이다.
"기술 부채"가 아닌 "기술 자산"의 전형적 예시다.

### Fix-and-Break는 본질적 한계

AI 코드 생성의 가장 큰 과제는 **전체 맥락 유지**다.
한 부분을 수정할 때 다른 부분과의 의존성을 완벽하게
파악하지 못하는 것은 현재 LLM의 구조적 한계이며,
프로젝트가 커질수록 이 문제는 심화된다. 이는 Bolt뿐
아니라 모든 AI 코딩 도구가 직면한 근본 문제다.

### 솔로 개발자 시대의 가속

Bolt.new가 증명하는 것은 1인 개발자가 이전에는
팀 단위로만 가능했던 결과물을 만들 수 있다는 점이다.
다만 "만들 수 있다"와 "유지보수할 수 있다"는 별개의
문제다. 프로토타입에서 프로덕션으로 넘어가는 순간
전통적 엔지니어링 역량이 여전히 필요하다.
