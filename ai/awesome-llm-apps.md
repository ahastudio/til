# Awesome LLM Apps

<https://github.com/Shubhamsaboo/awesome-llm-apps>

RAG, AI 에이전트, 멀티 에이전트 팀, MCP, 음성 에이전트
등을 활용한 LLM 앱 모음.
OpenAI, Anthropic, Google, xAI, Qwen, Llama 등
다양한 모델을 사용한 실용적인 구현 예제를 제공한다.
GitHub 스타 98.4k 이상으로 커뮤니티 검증을 받았다.

## 인사이트

### 학습 경로가 설계되어 있다

Starter → Advanced → Multi-Agent 순서로
자연스러운 학습 흐름을 제공한다.
여행 에이전트 하나만 봐도 로컬 버전, 클라우드 버전,
메모리 탑재 버전, MCP 버전, 멀티 에이전트 팀 버전이
단계별로 존재한다.
하나의 주제를 깊이 파고들며 난이도를 올릴 수 있다.

### LLM 앱의 주요 패턴이 보인다

프로젝트들을 관통하는 핵심 패턴은 다음과 같다.

- **RAG**: 외부 지식 검색 후 생성 (19개 프로젝트)
- **에이전트**: 도구를 사용하는 자율적 LLM (30개 이상)
- **멀티 에이전트**: 역할 분담하는 에이전트 팀 (13개)
- **MCP**: 모델 컨텍스트 프로토콜 통합 (4개)
- **음성**: 음성 입출력 에이전트 (4개)

RAG가 가장 기본이고,
에이전트가 주류이며,
멀티 에이전트가 최신 트렌드다.

### 비용 최적화 도구가 인상적이다

TOON 포맷으로 LLM API 비용을 30~60% 절감하고,
Headroom 컨텍스트 압축으로 50~90%까지 줄일 수 있다.
프로덕션 환경에서 LLM 비용은 핵심 과제인데,
이런 최적화 기법을 예제와 함께 제공하는 점이 실용적이다.

### 프레임워크 크래시 코스가 있다

Google ADK와 OpenAI Agents SDK에 대한
단계별 학습 자료를 포함한다.
에이전트 생성, 함수 호출, 구조화된 출력,
메모리, 콜백, 멀티 에이전트 패턴까지
프레임워크별로 체계적으로 다룬다.
새로운 에이전트 프레임워크를 빠르게 익히기 좋다.

### 모델에 구애받지 않는다

같은 앱을 OpenAI, Claude, Gemini, Llama 등
다양한 모델로 구현한 예제가 있다.
특정 모델에 종속되지 않는 설계를 배울 수 있고,
모델 간 비교도 가능하다.
