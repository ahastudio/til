# tts-bench

<https://github.com/5uck1ess/tts-bench>

## 소개

`tts-bench`는 로컬에서 구동 가능한 TTS(Text-to-Speech) 모델
55종을 여러 하드웨어 환경에서 비교 평가하는 오픈소스
벤치마크 도구다.
CPU, CUDA, Apple Silicon 등 서로 다른 플랫폼에서의 성능
차이를 객관적으로 재현 가능하게 비교하려는 목적으로
만들어졌다.

## 아키텍처

평가는 세 가지 관점으로 이뤄진다.

**속도(Speed)**: 콜드/웜 상태의 TTFA(time to first audio),
RTF(real-time factor), CPU·CUDA·Apple Silicon에서의 메모리
사용량을 측정한다.

**청취(Listen)**: 모든 모델에 대해 기본 음성과 보이스
클로닝 결과를 인라인으로 재생할 수 있는 갤러리를 제공한다.

**점수(Scores)**: UTMOS(자연스러움), WER(명료도), SIM(클로닝
충실도) 세 지표로 모델을 정량 평가한다.

41개 모델이 세 가지 형식의 참조 음성 클립으로부터 제로샷
보이스 클로닝을 지원하며, Supertonic·VoxCPM·OmniVoice 등
한국어를 포함한 다국어 지원 모델도 여럿 포함돼 있다.
보이스 클로닝 성능은 블라인드 A/B 투표 아레나를 통해
사람이 직접 순위를 매긴다.
OmniVoice가 음색 매칭에서 가장 높은 선호도를 얻었고,
Echo-TTS와 IndexTTS-2가 그 뒤를 이었다.

## 사용법

Python 3.11과 `uv` 패키지 매니저가 필요하다.

```bash
# Windows
.\install.ps1
python bench.py

# macOS/Linux
./install.sh
python bench.py

# 특정 모델만 설치
./install.sh kokoro piper miso
```

체감 테스트와 비교에는 별도 스크립트를 쓴다.

```bash
python speak.py kokoro
python compare.py "phrase"
```

2026년 6월 기준 성능 요약은 다음과 같다.

- CPU(Ryzen 9 9950X3D): Piper가 웜 TTFA 107ms, RTF 59배로
  최고 속도
- CUDA(RTX 5090): Kokoro가 웜 TTFA 67ms, RTF 104배로 최고
  속도
- CPU+MPS(Apple M4): Piper가 웜 TTFA 208ms, RTF 32배로
  최고 속도

디스크 용량은 모델 실행 환경에 약 39GB, 가중치 파일에 약
125GB로 총 165GB가 필요하다.

## 분석

TTS 모델 생태계는 최근 1~2년 사이 로컬 구동이 가능한
오픈 웨이트 모델이 급증하면서, "어떤 모델이 내 하드웨어에서
가장 빠르고 자연스러운가"라는 질문에 답하기가 오히려
어려워졌다.
`tts-bench`는 이 문제를 속도·청취·점수라는 세 축으로 나눠
정리함으로써, 주관적 청취 평가와 객관적 지표(UTMOS, WER,
SIM)를 함께 제시하는 접근을 취한다.
특히 보이스 클로닝 성능을 블라인드 A/B 투표로 측정한 점은,
자동화된 지표만으로는 포착하기 어려운 인간의 음색 선호를
반영하려는 시도로 볼 수 있다.

## 비평

55개 모델, 165GB에 달하는 가중치를 모두 내려받아 비교해야
한다는 점은 개인 개발자가 이 벤치마크를 직접 재현하기에는
상당한 진입 장벽이다.
또한 성능 요약이 특정 하드웨어 세 종류(Ryzen 9 9950X3D,
RTX 5090, Apple M4)에 한정돼 있어, 그보다 낮은 사양의
환경에서는 결과가 어떻게 달라지는지 가늠하기 어렵다.
프로젝트가 183개 스타 규모로 아직 초기 단계인 만큼, 모델
목록과 벤치마크 결과가 최신 모델 출시 속도를 얼마나
꾸준히 따라갈 수 있을지도 지켜볼 부분이다.

## 인사이트

로컬 TTS 생태계가 성숙해지면서, 이제 필요한 것은 새로운
모델 자체보다 기존 모델들을 공정하게 비교할 수 있는
공통 벤치마크 인프라라는 점을 이 프로젝트는 잘 보여준다.
속도·자연스러움·클로닝 충실도라는 세 축을 분리해 측정하는
방식은, 사용자가 "가장 좋은 모델"이 아니라 "내 하드웨어와
용도에 가장 적합한 모델"을 고를 수 있게 돕는다는 점에서
실용적이다.
