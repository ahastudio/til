# Maek, 로컬 우선 AI 메모리 워크스페이스

<https://maek.cognica.io/>

Show GN: [Lucene search core 기여 팀이 만든 Local-first AI Memory | GeekNews](https://news.hada.io/topic?id=31259)

## 소개

Maek은 대화 맥락이 세션마다 초기화되는 일반 챗봇과 달리, 세션을
넘나들며 맥락을 유지하는 macOS AI 어시스턴트다. Apache Lucene
10.5.0에 베이지안 BM25(BB25) 확률적 하이브리드 검색을 기여한
cognica 팀이 만들었으며, 반복적으로 같은 맥락을 다시 설명해야
하는 문제를 해결하는 데 초점을 맞췄다.

핵심은 이전 대화를 프롬프트에 그대로 다시 밀어넣는 방식이 아니라,
질의마다 관련 맥락을 동적으로 재구성한다는 데 있다. 이렇게 함으로써
컨텍스트 윈도우 낭비, 노이즈 누적, 중요한 정보 유실을 피한다.
아키텍처는 다섯 가지 신호를 결합한다. 키워드·벡터로 검색 가능한
채팅 메시지 이력, 근거로 인용되는 로컬 저장 문서 청크, 사람·조직·
결정 같은 개체를 추출한 관계 그래프, 압축된 상황 요약인 대화
상태, 그리고 BM25·벡터·그래프 연결·최신성을 확률적으로 종합하는
증거 기반 하이브리드 검색이다.

"Keep, Know, Own"이라는 세 가지 핵심 가치를 내세운다. Keep은
세션을 넘어 이어지는 대화 지속성, Know는 맥락의 깊은 의미를
분석하는 능력, Own은 모든 데이터를 기기에 로컬로 저장하고
선택적으로 종단간 암호화 동기화를 지원하는 완전한 데이터
프라이버시를 뜻한다. 응답이 어떤 메시지·문서·그래프·상태
정보로부터 만들어졌는지 사용자가 직접 확인할 수 있는 "검사
가능성(inspectability)"도 특징이다. ChatGPT Memory 같은 서비스
종속형 메모리 기능과 달리 독립 실행형 도구로 동작하며, 로컬
모델과 OpenAI 같은 서드파티 서비스를 함께 지원한다. 24단어 복구
구문으로 계정 접근을 관리하며, 팀은 개념의 명확성, 기술적
견고함, 기존 메모리 기능과의 차별성, 데이터 투명성, 초기 사용자
경험에 대한 피드백을 요청하고 있다.

## 분석

### "검사 가능성"이 신뢰의 핵심 축이다

AI 메모리 기능에 대한 가장 흔한 불안은 "내가 말한 적 없는
내용을 기억하고 있다고 착각하는" 환각이다. Maek이 응답의 근거가
된 메시지·문서·그래프 요소를 사용자에게 직접 노출하는 설계는,
메모리 시스템의 정확성을 사용자가 스스로 검증할 수 있게 하는
장치다. 이는 단순히 "기억한다"는 기능을 넘어, 그 기억이 왜
그렇게 구성됐는지를 투명하게 보여줘야 신뢰를 얻을 수 있다는
판단을 반영한다.

## 비평

### 다섯 신호를 결합하는 복잡도가 실제로 체감되는 이점으로 이어지는지 불분명하다

키워드·벡터·그래프·상태 요약·최신성을 확률적으로 종합한다는
아키텍처는 이론적으로 정교하지만, 이 복잡도가 단순한 벡터
검색 대비 얼마나 나은 결과를 내는지에 대한 정량적 근거는 제시되지
않았다. 다섯 가지 신호를 균형 있게 다루는 일은 그 자체로 튜닝
난이도가 높은 작업이며, 잘못 조정되면 오히려 노이즈가 늘어날
위험도 있다.

### 초기 공개 단계로, 커뮤니티 검증이 아직 부족하다

이 프로젝트는 아직 피드백을 구하는 단계이며, 외부 코멘트나
토론이 확인되지 않았다. 로컬 우선과 데이터 소유권이라는 매력적인
가치 제안이 실제 사용자 경험에서 어떻게 검증되는지는 시간이
더 지나야 판단할 수 있다.

## 인사이트

### 검색 기술 배경을 가진 팀이 메모리 문제를 검색 문제로 재정의한다

cognica 팀이 정보 검색(BB25를 Apache Lucene에 기여한 이력)
배경을 가지고 AI 메모리 제품에 뛰어들었다는 점은 시사하는 바가
있다. 많은 AI 메모리 제품이 "요약과 재주입"이라는 단순한 접근에
머무는 반면, 검색 전문성을 가진 팀은 메모리 문제를 본질적으로
"어떤 순간에 어떤 정보를 검색해 노출할 것인가"라는 정보 검색
문제로 재정의하고 있다. 이는 LLM 애플리케이션의 다음 경쟁
지점이 모델 자체가 아니라 그 모델에 무엇을 어떻게 공급하는지를
결정하는 검색·랭킹 계층으로 옮겨가고 있음을 보여주는 사례다.
