# IntelliJ IDEA

## Install

```bash
brew install intellij-idea-ce
```

## Command-line interface

<https://www.jetbrains.com/help/idea/working-with-the-ide-features-from-command-line.html>

```bash
mkdir -p ~/bin
touch ~/bin/idea
```
