# Ki Editor

> Multi-cursor combinatoric modal editor

<https://ki-editor.org/>

Rust로 개발된 터미널 기반 텍스트 편집기.
Neovim 4년 사용 경험에서 비롯된 "설정 지옥"에 대한 반발로
단일 개발자가 2023년에 시작한 프로젝트.

## 핵심 철학

### 위치 기반 키맵 (Positional Keymaps)

Vim의 `p` = paste 같은 니모닉 방식을 버리고
**물리적 위치**를 기준으로 키를 배치한다.

- 키보드 레이아웃 무관 (QWERTY/Dvorak/Colemak 동일하게 동작)
- 자주 쓰는 키를 홈로우에 배치 (사용 빈도 기반 최적화)
- Shift/Alt/Menu 레이어에서 같은 위치는 유사한 의미

### 선택 모드 중심 워크플로우

- Vim: 액션 → 이동 (e.g., `dw`)
- Helix: 이동 → 액션 (e.g., `w d`)
- Ki: **선택 모드 결정 → 이동 → 액션**

선택 모드만 바꾸면 이동 키가 통일된다.
Vim에서 Word→`w/b`, 줄→`j/k`, Git hunk→`]c/[c`처럼
제각각인 이동 키가 Ki에서는 모두 `j`/`l` 하나로 통일된다.

### 최소 설정 원칙

테마, 언어별 포매터, Tree-sitter 문법, LSP 정도만 설정 가능.
의도적으로 설정 가능 범위를 제한한다.

## 선택 모드 (Selection Modes)

### 연속 선택

- Syntax Node (Coarse / Fine)
- Line (Trimmed / Full)
- Word, Big Word, Subword
- Character

### 비연속 선택

- 검색 결과
- LSP 진단
- Git Hunk

## 10가지 표준 이동

모든 선택 모드에서 동일한 이동 키를 사용한다.

| 이동            | 의미                |
|-----------------|---------------------|
| Left / Right    | 의미 있는 이전/다음 |
| Previous / Next | 빈칸 없이 이전/다음 |
| Up / Down       | 위/아래 줄          |
| First / Last    | 처음/끝             |
| Jump            | 화면 내 EasyMotion  |

## 구조적 편집 (Tree-sitter 기반)

다른 에디터에서 플러그인이나 부가 기능인 AST 기반 편집이
Ki에서는 핵심 이동/편집의 기반이다.

- 커서 위치의 구문 노드 단위 선택/삭제
- 삭제 시 쉼표 자동 처리 (JSON, 함수 인자 등)
- 복사/붙여넣기 시 구분자 자동 추가
- **Swap**: 인자 순서 교환
- **Raise**: 부모 노드를 현재 자식 노드로 교체
  - `f(x)` → `x` 처럼 구문 오류 없음을 보장

## "모든 것이 에디터"

Helix가 GUI 컴포넌트 방식으로 돌아선 것과 달리,
프롬프트, 자동완성 드롭다운까지 동일한 에디터 컴포넌트.
컴포넌트마다 별도 키맵을 배울 필요가 없다.

## 기본 내장 기능

- LSP
- 파일 탐색기
- 전역 검색/교체
- 멀티커서 (구문 인식 편집 동시 적용)
- 외부 파일 변경 자동 감지

## 에디터 비교

| 항목              | Ki     | Vim/Neovim   | Helix    |
|-------------------|--------|--------------|----------|
| 구조적 편집       | 1급    | 플러그인     | 2급      |
| 멀티커서          | 내장   | 플러그인     | 내장     |
| 파일 탐색기       | 내장   | 내장(불안정) | 없음     |
| 전역 검색/교체    | 내장   | 설정 필요    | 교체없음 |
| 외부 파일 감지    | 있음   | 설정 필요    | 없음     |
| 키바인딩 일관성   | 높음   | 낮음         | 낮음     |

## 기술 스택

- Rust 69%, Tree-sitter Query 26%
- VS Code / JetBrains 플러그인 지원
- 라이선스: MPL-2.0
