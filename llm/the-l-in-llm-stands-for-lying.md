# The L in "LLM" Stands for Lying

[The L in "LLM" Stands for Lying — Acko.net](https://acko.net/blog/the-l-in-llm-stands-for-lying/)

[The L in "LLM" Stands for Lying | Hacker News](https://news.ycombinator.com/item?id=47257394)

Steven Wittens(acko.net 운영자)가 LLM의 본질을 "환각(Hallucination)"이
아닌 **"위조(Forgery)"**로 재프레이밍한 글.

## 핵심 주장: LLM 출력 = 위조품

할루시네이션이라는 용어는 LLM의 실수를 마치 인지적 오류처럼 포장하지만,
저자는 더 정직한 단어를 제안한다: **위조(forgery)**.

> 반 고흐의 서명이 있지만 그가 그리지 않은 그림은 위조다.
> 동의를 날조하는 법적 문서는 위조다.
> 데이터를 만들고 인용을 조작하는 연구는 위조다.

LLM은 개인이 자신의 잠재적 결과물 — 혹은 타인의 것 — 을 실제로
생산하는 것보다 빠르게 **위조**할 수 있게 해준다.

위조 자체는 픽션이나 자기 표현으로서는 합법이다. 문제는 위조품을
진품의 대체물로 사용하려 할 때 발생한다.

## AI 하이프에 대한 회의론

수년간의 LLM 기반 툴링에도 불구하고 소프트웨어는 "예전과 거의
다를 바 없이 생김새, 느낌, 기능이 비슷하다: 간신히."

새 모델은 이미 은퇴한 모델이 이루기로 했던 약속을 달성하기 위해
계속 훈련된다. 하이프는 투자를 끌어들이고, 투자는 더 많은 하이프를
정당화하는 데 사용된다.

> "AI를 사용하지 않아도 괜찮다." 그렇다고 기술 세계에서 뒤처지는
> 게 아니다.

## 바이브 코딩과 오픈소스 파괴

바이브 코딩(vibe coding)이 정당화되려면 엔지니어들이 자신의 결과물을
**일회용**이고, 비창의적이며, 크레딧을 받을 자격이 없다고 봐야 한다.

오픈소스 메인테이너들은 이미 동기부여된 기여자를 찾기 힘든데,
"GitHub 이력을 만들기 위해 부정행위를 하려는 기여자들의 슬롭 코드
PR"까지 처리하게 됐다.

일부 프로젝트는 공개 기여를 닫고 버그 바운티를 폐지했다.

**2026년 기준:** 새 직원이 상세한 설명과 주석이 달린 PR을 제출하면
모든 내용을 의심하라.

## 슬롭 코드(Slop Code) 문제

LLM이 생성한 코드는 버그 투성이의 **슬롭 코드**로 불린다.
Microsoft Copilot Discord에서 "Microslop"이라는 비속어를 금지했고,
사용자 반발은 "스팸" 혹은 "해로운 것"으로 프레이밍됐다.

약속이 실제 결과물보다 훨씬 더 가치 있다는 것을 방증한다.

## 해결책: 소스 어트리뷰션

저자가 제시하는 유일한 해결책:

> LLM이 추론과 함께 **올바른 소스 어트리뷰션**을 수행해야 한다.

이렇게 하면:

- 예술 저작권 문제 해결에 도움
- 바이브 코드가 얼마나 기존 코드베이스에서 원저자, 라이선스,
  링크를 생략한 채 복붙된 것인지 드러남

어떤 법원도 AI 출력이 합법인지 저작권이 있는지 판결을 내릴 수
없다. 왜냐하면 출처가 없기 때문이다. **AI 출력은 달리 증명되지
않는 한 위조품으로 취급해야 한다.**

## 인사이트

| 개념                   | 기존 프레이밍     | 저자의 재프레이밍       |
| ---------------------- | ----------------- | ----------------------- |
| LLM 오류               | 환각(Hallucination) | 위조(Forgery)         |
| 바이브 코딩            | 생산성 향상       | 책임감 외주, 크레딧 도용 |
| AI 코드 기여           | 민주화            | 오픈소스 생태계 파괴    |
| 빠른 온보딩            | 역량 향상         | 무능을 봇에 전가        |

**핵심 인사이트:** "할루시네이션"은 LLM의 실수를 인지적 한계로
중립화하지만, "위조"는 **의도와 책임**의 언어다. 프레이밍의 전환
자체가 강력한 비판이다.

LLM 시대의 진짜 문제는 AI가 틀린다는 게 아니라, 진품처럼 보이는
모조품이 **진품을 대체**하도록 허용하는 사회적 구조다.
